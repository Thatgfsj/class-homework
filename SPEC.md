# 中文233班作业发布网站 - 设计规格

## 1. Concept & Vision

一个温暖而现代的学术作业发布平台，融合中式文化美学与现代极简设计。网站如同一位贴心的学习伙伴，让每次查看作业都成为愉快的体验。采用「墨韵书香」设计理念——以传统水墨元素为灵感，配合现代卡片式布局，既有文化底蕴又不失年轻活力。

## 2. Design Language

### 美学方向
- **风格**: 新中式极简 + 学术清新
- **参考**: 传统书房与现代咖啡馆的融合
- **氛围**: 温暖、宁静、有书卷气

### 色彩系统
```css
:root {
  /* 主色 - 墨青色 */
  --primary: #1a3a4a;
  --primary-light: #2d5a6b;
  --primary-dark: #0f2a38;

  /* 强调色 - 朱砂红 */
  --accent: #c75555;
  --accent-light: #e07878;
  --accent-dark: #a04545;

  /* 背景色 */
  --bg-primary: #faf8f5;      /* 宣纸白 */
  --bg-secondary: #f0ebe4;    /* 淡米色 */
  --bg-card: #ffffff;
  --bg-dark: #1a1a1a;

  /* 文字色 */
  --text-primary: #2c2c2c;
  --text-secondary: #666666;
  --text-muted: #999999;

  /* 边框与分割 */
  --border: #e5e0d8;
  --border-dark: #3a3a3a;

  /* 渐变 */
  --gradient-warm: linear-gradient(135deg, #faf8f5 0%, #f0ebe4 100%);
  --gradient-accent: linear-gradient(135deg, #c75555 0%, #a04545 100%);
}
```

### 字体
- **标题**: "Noto Serif SC" (思源宋体) - 体现中文书卷气
- **正文**: "Noto Sans SC" (思源黑体) - 清晰易读
- **英文/数字**: "Crimson Pro" - 优雅衬线体
- **备选**: system-ui, sans-serif

### 间距系统
- 基础单位: 8px
- 间距层级: 8, 16, 24, 32, 48, 64, 96px
- 卡片圆角: 16px
- 按钮圆角: 8px
- 最大内容宽度: 1200px

### 动效哲学
- **进入动画**: 卡片从下方淡入上移，错开100ms
- **悬停效果**: 轻微上浮 + 阴影加深，200ms ease-out
- **页面过渡**: 内容区域渐显，400ms
- **微交互**: 按钮点击缩放0.98，图标旋转/摇摆

### 响应式断点
- Mobile: < 640px (单列)
- Tablet: 640px - 1024px (双列)

### 视觉节奏
- Hero区域: 大量留白，柔和渐变背景
- 作业卡片: 紧凑但透气，信息密度适中
- Footer: 低调收尾

## 4. Features & Interactions

### 核心功能
1. **作业展示**: 按发布日期倒序显示所有作业
2. **作业详情**: 点击展开查看完整内容
3. **筛选功能**: 按科目筛选作业
4. **搜索功能**: 关键词快速搜索作业
5. **截止提醒**: 显示距截止日期的倒计时

### 交互细节
- **卡片悬停**: 上浮4px + 阴影扩展 + 边框微亮
- **点击展开**: 卡片展开动画，显示完整作业描述
- **筛选切换**: 平滑过渡，未选中卡片淡出
- **搜索输入**: 实时过滤，0.3s防抖
- **空状态**: 显示友好插图 + "暂无作业"文案

### 状态处理
- **加载中**: 骨架屏动画 (渐变闪烁)
- **无结果**: 插图 + "未找到匹配的作业"
- **作业过期**: 灰色遮罩 + "已截止"标记

## 5. Component Inventory

### Header
- 左侧: 班级Logo (毛笔字"文"图形)
- 中间: "中文233" 大标题
- 右侧: 学期下拉选择器
- 背景: 半透明白 + 毛玻璃效果
- 固定顶部，滚动时保持

### Hero Section
- 大标题: "作业公告板"
- 副标题: 当前日期 + 星期
- 装饰: 水墨风格云纹/山水剪影 (SVG)
- 背景: 暖色渐变 + 微妙纹理

### 作业卡片 (HomeworkCard)
- **默认状态**:
  - 白色背景，柔和阴影
  - 左侧彩色边条 (按科目着色)
  - 科目标签 + 作业标题 + 发布日期
  - 简要描述 (最多2行)
  - 截止日期 + 状态指示

- **悬停状态**:
  - 上移4px
  - 阴影加深扩大
  - 边框微亮

- **展开状态**:
  - 卡片高度扩展
  - 显示完整描述
  - 附件下载按钮 (如有)
  - 相关链接

- **已截止状态**:
  - 整体灰度降低
  - 右上角显示"已截止"标签

### 科目标签 (SubjectTag)
- 颜色编码:
  - 语文: #c75555 (朱砂红)
  - 数学: #4a7c59 (竹青)
  - 英语: #5a7c9a (靛蓝)
  - 物理: #7c5a7c (紫藤)
  - 化学: #7c6a4a (土黄)
  - 历史: #6a5a4a (赭石)
  - 其他: #666666 (石墨)

### 筛选栏 (FilterBar)
- 科目筛选按钮组
- 搜索输入框 (带放大镜图标)
- 当前筛选状态提示

### Footer
- 班级名称
- 最后更新时间
- 技术支持信息
- 简洁分隔线

## 6. Technical Approach

### 技术栈
- **纯静态**: HTML5 + CSS3 + Vanilla JavaScript
- **无框架依赖**: 轻量化，快速加载
- **GitHub Pages**: 免费托管
